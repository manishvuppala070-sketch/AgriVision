This is a sample directory structure for the EuroSAT dataset.

To use this project:
1. Download the full EuroSAT dataset from: https://github.com/phelber/EuroSAT
2. Extract it and rename the folder to "EuroSAT"
3. Replace this EuroSAT_sample folder with the full EuroSAT dataset

The full dataset contains 10 classes:
- AnnualCrop, Forest, HerbaceousVegetation, Highway, Industrial
- Pasture, PermanentCrop, Residential, River, SeaLake
